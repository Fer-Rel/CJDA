# 💰 Sistema de Ahorros - CJDA

Este es un sistema de gestión para el control de ahorros de clientes, diseñado con un enfoque modular.

## 🚀 Tecnologías utilizadas
* **Frontend:** React + Vite
* **Backend:** Supabase (Autenticación y Base de Datos)
* **Estilos:** Tailwind CSS / CSS3

## 🛠️ Funcionalidades (Módulo 1)
* **Autenticación segura:** Login con persistencia de sesión.
* **Gestión de Clientes:** CRUD completo y búsqueda en tiempo real.
* **Vouchers Térmicos:** Generación de tickets optimizados para impresoras Epson (TM-T20III).

## 📦 Instalación
1. Clonar el repositorio.
2. Ejecutar `npm install`.
3. Configurar las variables de entorno basadas en `.env.example`.
